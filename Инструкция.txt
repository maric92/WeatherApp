Приложение отображает по запросу температуру(в Цельсиях), также добавляется картинка, отражающаяб какая погода в городе сейчас.
Данные берутся с сайта https://openweathermap.org/api, путем привязки личного ключа и выдачи информации по запросу(в коде все видно будет)
Запуск - открыть любой файл проекта, далее по стандарту - python manage.py runserver, на локальном сервере выдаст форму.
Так же, помимо поисковой строкиб можно найти и добавить на главную страницу все города через админа http://127.0.0.1:8000/admin/weather/city/
